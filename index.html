RLCraft Remastered

Download the complete modpack without the need for Curseforge.

Read More Visit RLCraft Remastered

Install Instructions

  • Download the above modpack
  • Unzip it and place the rlcraft directory in %appdata%\.minecraft in Windows or ~\Library\Application Support\ folder on Mac (Cmd+Shift+G)
  • Download and run the Forge 1.12.2 installer (you will need to add any previous Forge Minecraft profiles)
  • Open Minecraft and Edit the Forge 1.12.2 profile, change the Game Directory to the directory above, click More Options and make sure that in JVM Arguments, you have -Xmx4G and not any other -Xmx value
